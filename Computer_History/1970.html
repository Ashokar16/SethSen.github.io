Computer History

A timeline that shows the history of computers from 1970's to all the way to present day.

1970's 1980's 1990's 2000's 2010's

Hewlett-Packard introduced the HP-35

HP created a scientific calculator that can fit all of the features of their desktop scientific calculator while being small enough to fit in a pocket. The calculator was the first calculator to solve logarithmic and trigonometric functions.

C Programming language is released

Dennis Ritchie and his team created the program C and it was based on the earlier language BCPL (Basic Combined Programming Language) and soon after re-wrote the source code for Unix in C. As such, Unix was easily ported to other computers and spread swiftly.

IBM scamp is developed

Dr. Paul Friedl the engineer created the Special Computer APL Machine Portable (SCAMP). It is a portable computer designed to run the APL programming language which had a CRT display, and cassette tape storage. It only took six months to develop.

Xerox PARC Alto introduced

The Alto is a groundbreaking computer. It had a graphical user interface using windows, icons, and a mouse, and worked together with other Altos. It could also share files and print out documents. Alto also included The Applications like a WYSISYG word processor known as “Bravo,” a paint program, a graphics editor, and email.

Apple II is introduced

The Apple II was Sold a main logic board, switching power supply, keyboard, case, manual, game paddles, and cassette tape. When connected to a color television set, the Apple II produced brilliant color graphics. It was very popular and spread throughout schools and many homes.
