Computer History

A timeline that shows the history of computers from 1970's to all the way to present day.

1970's 1980's 1990's 2000's 2010's

Sinclair ZX80 introduced

This very small home computer is available for £79 or pre-assembled for £99. Inside was a Z80 microprocessor. It was displayed on the user’s home TV screen through use of an adapter. About 50,000 were sold in Britain, primarily to hobbyists.

3 1/2-inch floppy drive

Sony introduces the first 3 ½-inch floppy drives in 1981. The first significant company to adopt the 3 ½-inch floppy for general use was Hewlett-Packard in 1982. It established momentum for the format and which helped standardized the microfloppy.

Microsoft introcudes Word

Microsoft announces Word, originally called Multi-Tool Word. Microsoft distributed 450,000 disks containing a demonstration version of its Word program in the November issue of PC World magazine, giving readers a chance to try the program for free. It competed with WordPerfect for market share as a word processing program, and it was not until Microsoft Word for Windows was introduced in 1989 that it became a global standard.

C++ Programming language is introduced

The C++ programming language is a very popular language in the computer industry. Stroustrup, from AT&T Bell Labs, said his motivation stemmed from a desire to create a language that would allow for more complex programs and which combined the low-level features of BCPL with the high-level structures of Simula.

Computer defeats master chess player

David Levy is the first master chess player to be defeated by a computer. The program Deep Thought defeats Levy who had beaten all other previous computer counterparts since 1968.
